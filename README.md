# Heart Disease Data Visualization Practice

## Overview
This repository contains a visualization practice project focused on mastering 10 essential chart types using Matplotlib and Seaborn.
The goal is to create clear, interpretable, and publication-quality visualizations, not to build predictive models.

## Dataset
This project uses the **Heart Disease Dataset** from Kaggle.

Due to licensing restrictions, the dataset is not included in this repository.

You can download it from:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

## Tools & Libraries
- Python
- Matplotlib
- Seaborn
- Pandas
- NumPy
- Google Colab
